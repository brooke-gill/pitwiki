Drop confirmation improvements
From the Hypixel Pit Wiki

Drop confirmation improvements was a minor version of the Hypixel Pit. It was an intermediary update between the Pizza Patch and the release of the Corals map.

The update simply added two features:

  • Added the /dropconfirm on and /dropconfirm off commands, which toggle whether a confirmation message appears when attempting to drop rare items.
  • Players can now press the drop key three times to confirm dropping an item.

One bug was fixed in this update:

  • Fixed a visual bug when dropping an item where the item wouldn't drop but wouldn't be visible in the user's hotbar.
